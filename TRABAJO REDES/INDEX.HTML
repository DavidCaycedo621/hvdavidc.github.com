Organización jerárquica de protocolos de redes de acuerdo con los niveles del modelo OSI

Capítulo 1: Protocolo de Internet (IP)

Descripción del IP

El Protocolo de Internet (IP) es la base de todas las comunicaciones en Internet. IP es responsable de entregar paquetes de datos de la fuente al destino.

Capítulo 2: Protocolo de Control de Transmisión (TCP)

Descripción del TCP

El Protocolo de Control de Transmisión (TCP) se construye sobre IP y proporciona una conexión confiable entre dos hosts en Internet. TCP se encarga de controlar el tamaño y la velocidad de la transmisión de datos.

Capítulo 3: Protocolo de Transferencia de Hipertexto (HTTP)

Descripción del HTTP

El Protocolo de Transferencia de Hipertexto (HTTP) se utiliza para transferir datos a través de la web. HTTP se construye sobre TCP y permite la transferencia de archivos de texto, imágenes y otros datos entre el servidor web y el cliente.

Capítulo 4: ¿Por qué necesitamos protocolos de red?

Descripción de la necesidad de los protocolos de red

Los protocolos de red son conjuntos de reglas establecidas que dictan cómo formatear, transmitir y recibir datos para que los dispositivos de red de computadoras, desde servidores y enrutadores hasta terminales, puedan comunicarse independientemente de las diferencias en sus infraestructuras subyacentes, diseños o estándares.
